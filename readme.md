じゃんけんアプリ（php版）

## DEMO

  

## 紹介と使い方

・phpの書き込み（自分が出したいものを選択）
・phpの読み込み（コンピューターとじゃんけんで対決）
・上記書き込んだもの（選択したもの）を受取側で表示
・if関数を使いじゃんけん対決
・コンピューターとの勝敗を表示



## 工夫した点
